# Food Roulette - 今天吃什么大转盘

## 项目简介

Food Roulette 是一个命令行交互式的“今天吃什么”随机选择工具。当您为午餐或晚餐的选择感到困扰时，本程序通过模拟大转盘的动画效果，从预设的候选菜单中随机选择一个食物，为您的用餐决策增添趣味性。

## 核心特性

- **交互式命令行界面**：简洁直观的终端交互体验。
- **动态转盘动画**：模拟真实转盘转动，速度由快至慢，最终定格结果。
- **预设候选菜单**：包含火锅、烧烤、麦当劳、肯德基等常见选项。
- **循环抽奖机制**：单次抽奖结束后可选择重新开始或退出。
- **跨平台兼容**：支持 Windows、macOS 和 Linux 操作系统。

## 环境依赖与安装步骤

### 环境要求

- **Python 3.6 或更高版本**
- 终端或命令行工具

### 安装步骤

1. **克隆或下载项目**
   ```bash
   git clone <repository-url>
   cd food_roulette
   ```
   或直接下载 `main.py` 文件。

2. **（可选）创建虚拟环境**
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS/Linux
   source venv/bin/activate
   ```

3. **运行程序**
   由于本项目为纯 Python 脚本，无需安装额外依赖，可直接运行。

## 快速启动命令

在项目目录下，执行以下命令：

```bash
python main.py
```

## 使用说明与运行示例

### 程序启动

运行程序后，您将看到如下界面：
```
==================================================
           今天吃什么大转盘
==================================================
候选菜单: 火锅, 烧烤, 麦当劳, 肯德基, 炒饭, 麻辣烫, 沙拉, 饿着, 袁记云饺
--------------------------------------------------

按回车键开始抽奖 (或输入 'q' 退出):
```

### 抽奖过程

1. 按下回车键开始抽奖。
2. 终端将显示转盘动画，候选菜单项会依次高亮显示，模拟指针转动。
3. 动画速度逐渐减慢，最终定格在一个食物上，并显示恭喜信息。

### 示例输出

抽奖结果示例：
```
==================================================
         ⭐ 最终结果: 火锅 ⭐
==================================================

🎉 今天就去吃: 火锅 ! 🎉

输入回车重新抽奖，输入 'q' 退出:
```

### 交互选项

- **开始抽奖**：在提示符下直接按回车键。
- **退出程序**：在开始前或抽奖后输入 `q` 并按回车。
- **重新抽奖**：抽奖结束后直接按回车键。
- **中断程序**：随时可按 `Ctrl+C` 中断当前操作或退出程序。

## 项目结构

项目为单文件脚本，所有功能均集成在 `main.py` 中：

- **`FoodRoulette` 类**：核心功能类，包含菜单管理、转盘动画、用户交互等方法。
- **`main()` 函数**：程序入口点，初始化并运行大转盘。

## 自定义菜单

如需修改候选食物列表，可编辑 `main.py` 文件中 `FoodRoulette` 类的 `__init__` 方法中的 `self.menu` 列表。

## 注意事项

1. 程序依赖终端清屏功能，在不同操作系统上均能正常工作。
2. 转盘动画的速度曲线可通过修改 `self.speed_factors` 列表调整。
3. 程序已处理常见的键盘中断（`Ctrl+C`）和输入错误。

## 许可证

本项目为示例项目，可自由使用和修改。
